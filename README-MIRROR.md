# Este repositório é um espelho

O desenvolvimento do Prumo acontece em [tharso/prumo-dev](https://github.com/tharso/prumo-dev).

Este repositório é atualizado automaticamente a cada push na main do repo de desenvolvimento. **Não abra PRs aqui** — eles serão sobrescritos no próximo espelhamento. Abra em `prumo-dev`.

Usuários finais: este é o repo correto pra instalar o Prumo via marketplace (Cowork, Claude Code), `pip install prumo-runtime`, ou clone direto.
