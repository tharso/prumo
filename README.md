# Prumo

**Micromanagement. Sem o idiota.**

Prumo é um sistema de organização de vida pessoal que transforma o Claude em interface única para capturar, processar, lembrar e cobrar. O nome vem de "estar no prumo" — alinhado, no eixo.

## O que faz

Três gestos. Sistema inteiro:

1. **"Bom dia"** → briefing do dia (compromissos, pendências, cobranças)
2. **Dump** → despeja qualquer informação a qualquer momento
3. **"Revisão"** → revisão semanal completa

O Claude vira seu co-piloto: captura tudo que entra na sua vida, organiza, e cobra quando algo fica parado. Funciona como um microgerente competente que não tem ego, não tem ansiedade, e ainda faz o trabalho sujo por você.

## Requisitos

- Claude no plano Max ([assinar aqui](https://claude.ai/upgrade))
- Claude Desktop com Cowork mode ([acessar aqui](https://claude.ai/cowork))

## Instalação

1. [Baixe o plugin (.zip)](https://github.com/tharso/prumo/archive/refs/heads/main.zip)
2. Abra o Claude Desktop → Cowork
3. Clique em **Plugins** (sidebar) → **"+"** → arraste o .zip
4. Na tela do Cowork, **antes de iniciar a conversa**, clique no ícone de pasta (abaixo e à esquerda da caixa de input) e selecione onde quer organizar seus arquivos
5. Digite **`/Prumo`** e o setup conversacional começa

Em ~14 minutos você tem tudo rodando.
