# Adapter Boundaries

Esta pasta guarda notas e limites do que e conscientemente especifico de host.

Se uma regra depende de:

1. plugin store;
2. slash command;
3. affordance grafica de um app;
4. shell path especifico;
5. permissao de app;
6. workaround historico de distribuicao;

ela nao mora no canon compartilhado. Mora aqui, em doc operacional de host, ou fica no runtime como implementacao.

O objetivo desta pasta nao e glorificar excecao. E impedir que excecao se fantasie de lei natural.
