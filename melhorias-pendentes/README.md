# Melhorias pendentes para o Prumo

Cada arquivo `.md` nesta pasta é um prompt pronto pra rodar na janela do projeto Prumo.

**Como usar:** Abra o projeto Prumo no Cowork, copie o conteúdo do prompt, cole no chat. O agente do Prumo aplica a melhoria no template.

**Depois de aplicar:** Delete o arquivo do prompt (ou mova pra uma subpasta `aplicados/` se quiser histórico).

**Origem:** Melhorias identificadas durante o uso diário do DailyLife/Claudia que são generalizáveis pro template do Prumo.
